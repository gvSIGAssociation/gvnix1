Introduction
===============

Add-on giving support for Spring MVC editors configuration.


Project contents
=================

This folder contains add-on sources and documents folder ``docs`` with documentation of this project.


Installation Proof
===================

Install the Add-on.
--------------------

Install the Add-on in gvNIX/Roo shell with the required commands for 1.1.5-RELEASE version.

Create a web application
-------------------------

Create a web application with gvNIX/Roo Shell script::
    script clinic.roo


Run the add-on command:
    web mvc binding stringTrimmer
