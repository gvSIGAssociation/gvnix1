Local services creation and import and export remote web services.
  • service: Local services. Create classes and methods.
  • service import: Access web services as local services.
  • service export: Expose local methods and classes as web services.
  • service export ws: Expose local methods and clasess as web services generated from a WSDL definition
  • list: List local class methods and web service operations available.
  • security: Access secure servers WSDL and import web services with application certificate.
